# FE-06 异常与预警模块开发指南

## 1. 文档说明

- 本文件用于指导 `FE-06` 的独立开发。
- 开发前先读取前端并发清单、后端异常和预警接口文档。

## 2. 模块目标

- 实现异常列表、异常详情和预警列表页面，并展示 AI 判定依据与预警建议。

## 3. 模块边界

- 负责展示 AI 判定摘要、决策链和预警建议。
- 不提交复核结果。

## 4. 前置依赖

- `FE-01`
- 后端依赖 `BE-05`、`BE-06`

## 5. 固定接口清单

1. `GET /api/exception/list`
2. `GET /api/exception/{id}`
3. `GET /api/exception/{id}/decision-trace`
4. `GET /api/exception/{id}/analysis-brief`
5. `GET /api/warning/list`
6. `GET /api/warning/{id}/advice`

## 6. 固定字段与状态约定

- 异常列表展示类型、风险等级、处理状态。
- 异常详情需展示 AI 判定摘要、理由摘要和处理建议。
- 预警列表需能跳转关联异常详情，并展示 AI 预警摘要与优先级。

## 7. 代码目录范围

- `src/views/exception/`
- `src/views/warning/`
- `src/api/`

## 8. 详细开发步骤

1. 实现异常列表页。
2. 实现异常详情页。
3. 接入异常分析摘要和决策链接口。
4. 实现预警列表页。
5. 接入预警建议接口。
6. 打通从预警跳转到异常详情。
7. 补齐筛选、空态和错误态。

## 9. 最小验证命令

```bash
npm run build
```

## 10. 完成标准

- 异常与预警页面闭环。
- 详情和跳转链路清晰。
- AI 判定依据、决策链和预警建议可见。

## 11. 禁止改动项

- 不在本模块提交复核结果。

## 12. 文档同步要求

- 异常详情字段以后端 API 文档为准，不自行扩展。
