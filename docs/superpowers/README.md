# Superpowers 文档说明

本目录保存历史设计稿与实施计划，主要用于回溯前端壳层、认证集成和阶段性实现思路，不作为当前运行口径的唯一来源。

## 当前口径优先级

- 当前启动、测试、构建命令以仓库根 `README.md` 和 `package.json` 为准。
- 当前代理、环境变量和开发服务器行为以仓库根 `README.md`、`.env.example` 和 `vite.config.js` 为准。
- 当前路由、权限、状态和页面行为以 `src/router/`、`src/store/`、`src/views/`、`src/api/` 的实际实现为准。

## 阅读建议

- 需要查看历史方案、拆解过程和阶段性取舍时，阅读 `specs/` 与 `plans/`。
- 如果历史文档中的测试命令、页面结构或环境说明与当前仓库行为不一致，优先采用“当前口径优先级”中的文件，不逐篇回写旧记录。
